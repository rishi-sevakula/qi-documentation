---
title: Glossary
description: Key terms used in GEP Qi Studio.
---

**Agent orchestration**: A workflow-based agent built on the visual canvas.  
**Quantum agent**: A lightweight agent centered around direct LLM interaction.  
**Node**: A single step in a workflow graph.  
**Edge**: A connection that defines execution flow between nodes.  
**Domain context**: The client/domain/environment/app/module selection that scopes all APIs.  
**Design-time**: The configuration layer for agents and tools.  
**Runtime**: The execution layer for workflows and streaming events.  
**HITL**: Human-in-the-loop interrupts that require user input.  
**MCP**: Model Context Protocol, used for dynamic tool discovery.
